# 浩哥记账

## 项目概述

| 项目 | 说明 |
|------|------|
| **项目名称** | 浩哥记账 |
| **项目类型** | 跨平台桌面应用 |
| **目标平台** | Windows 10/11、macOS |
| **目标用户** | 普通个人用户（不懂编程） |
| **核心用途** | 记录日常个人支出，查看消费统计，管理个人财务 |

---

## 核心功能（MVP 版本）

### 1. 记账记录
- **添加支出**：填写金额（人民币/元）、选择分类、选择日期、添加备注（可选）
- **记录列表**：按时间倒序展示所有支出记录，显示金额、分类、日期、备注
- **编辑记录**：可修改已有记录的任意字段
- **删除记录**：可删除单条记录（建议添加确认提示）

### 2. 分类管理
- **二级分类**：先选择一级大类，再选择二级小类
- **预设分类**：系统内置一套完整的分类体系（见下方）
- **自定义分类**：用户可自行添加、修改、删除分类

### 3. 数据统计
- **月度汇总**：按月查看总支出金额
- **分类占比**：以饼图或条形图展示各一级大类的消费占比
- **筛选查询**：按分类、日期范围筛选记录

### 4. 数据存储
- **本地存储**：所有数据保存在用户本机，无需联网
- **数据安全**：数据文件存储在用户本地，不上传任何服务器

## 技术栈方案（待用户选择）

> ⚠️ 在开始编码前，需要用户从以下 3 个方案中选择一个。选择后，本文件将更新为选定方案。

### 方案 A：Electron + React + TypeScript

**技术说明**：Electron 是一个把网页打包成桌面应用的框架，React 是目前最流行的前端界面库，TypeScript 是带类型检查的 JavaScript。

| 维度 | 评价 |
|------|------|
| ✅ 生态成熟度 | 最成熟，社区最大，遇到问题容易找到解决方案 |
| ✅ UI 组件 | 可直接使用 Ant Design 等成熟的 UI 组件库，界面专业 |
| ✅ 跨平台 | Windows/Mac 完美支持，体验一致 |
| ✅ 学习资源 | 教程和文档极其丰富，中文资源多 |
| ❌ 安装包大小 | 较大，约 120-180 MB |
| ❌ 内存占用 | 较高，运行时约 150-300 MB |
| 🎯 **推荐场景** | 追求功能完善、UI 美观、不在意安装包大小 |

---

### 方案 B：Tauri + React + TypeScript

**技术说明**：Tauri 是新一代轻量桌面框架，利用操作系统自带的浏览器引擎渲染界面，后端用 Rust 语言，打包体积非常小。

| 维度 | 评价 |
|------|------|
| ✅ 安装包大小 | 极小，约 5-15 MB |
| ✅ 内存占用 | 低，运行时约 50-100 MB |
| ✅ 安全性 | 沙箱隔离，安全性更好 |
| ✅ 界面技术 | 同样使用 React，UI 组件库可复用 |
| ⚠️ 生态成熟度 | 较新（2022年发布），文档和社区在快速增长中 |
| ⚠️ Rust 后端 | 部分功能需要用 Rust 编写，学习门槛稍高 |
| 🎯 **推荐场景** | 追求轻量高效、注重安装包大小和运行性能 |

---

### 方案 C：Flutter 桌面版

**技术说明**：Flutter 是 Google 出品的跨平台 UI 框架，使用 Dart 语言，一套代码同时运行在移动端、Web 和桌面端。

| 维度 | 评价 |
|------|------|
| ✅ UI 一致性 | 跨平台 UI 完全一致，Material Design 风格美观 |
| ✅ 性能 | 接近原生性能，动画流畅 |
| ✅ 可扩展性 | 可轻松扩展到 iOS/Android 手机端 |
| ✅ Google 支持 | Google 持续投入开发，长期维护有保障 |
| ❌ 桌面端成熟度 | 桌面端支持仍在完善中（2023年稳定） |
| ❌ 中文社区 | 桌面端中文教程和资源相对较少 |
| ❌ 安装包 | 约 30-80 MB |
| 🎯 **推荐场景** | 追求原生性能体验，未来可能扩展到手机端 |

---

### 💡 个人推荐

> 对个人记账应用来说，**方案 B（Tauri + React）** 是最佳平衡：安装包小、运行轻量、界面专业、开发效率高。如果追求最快的开发速度和最丰富的 UI 组件，选 **方案 A（Electron + React）**。

## UI 设计参考

### 整体风格
- **设计方向**：简洁、清爽、现代，参考"鲨鱼记账"、"随手记"等主流记账应用
- **主色调**：暖色系（橙色 `#FA8C16` / 金色 `#F5A623`），体现财富、温暖感
- **辅助色**：深灰文字、浅灰背景、白色卡片
- **字体**：系统默认中文字体（苹方 / 微软雅黑）

### 页面布局
```
┌──────────────┬──────────────────────────────────┐
│              │                                  │
│   左侧导航    │         主内容区域                │
│              │                                  │
│  · 记一笔    │    （根据导航选项显示对应内容）     │
│  · 账单列表   │                                  │
│  · 月度统计   │                                  │
│  · 分类管理   │                                  │
│              │                                  │
└──────────────┴──────────────────────────────────┘
```

### 页面清单
1. **记一笔**：记账表单页（金额输入 + 分类选择 + 日期 + 备注）
2. **账单列表**：按时间倒序展示所有记录，支持筛选和搜索
3. **月度统计**：月度总支出数字 + 分类饼图 + 排行榜
4. **分类管理**：一级/二级分类的增删改
